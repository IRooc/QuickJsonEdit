## Quickly edit hierarchical and non-hierarchical json files

This tool simplifies editing specific values within JSON files by flattening their structure into selectable paths (e.g., `config/database/[0]/connectionString`).

## How to Run
1. Install the [.NET SDK](https://dotnet.microsoft.com/download).
2. Run `dotnet run` in the project root.
3. Open `https://localhost:7203` or `http://localhost:5240` in your browser.
